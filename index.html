Overview 🛡️

Welcome to Fadi's Assistant — a multipurpose Discord bot. This page explains what data we collect, why we collect it, how we store it, and your rights.

Data Collection 📥

  • User IDs
  • Server IDs
  • Channel IDs
  • Bot logs & configuration (optional per server)

Data is only collected to enable bot features like moderation, configuration persistence, and audit logs.

Data Storage 💾

Data is stored as JSON files within the bot's private storage (e.g., Railway’s /data/). Backups, if any, are handled securely and access is restricted to the developer.

How We Use Data ⚙️

We use data to maintain the bot, respond to moderation requests, and provide configuration persistence. We do not sell data or use it for third-party marketing.

Data Retention 🗄️

Default retention: configuration data is kept until removed by server admins or upon request. Logs older than 90 days may be purged automatically (configurable).

Transparency Report 🔍

Major privacy changes and incidents will be published here or via the developer's Discord. Contact the developer to request copies of any logs that pertain to you.
